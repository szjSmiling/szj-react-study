### 1.安装 npx,并且简单使用
```
npm i -D cowsay
npx cowsay hello
```
### 2.使用 npx 创建项目
```
npx create-react-app my-app
```
### 3.启动项目
```
yarn start
```
### 4. 安装使用
```
yarn add react-router-dom // 路由
yarn add node-sass --save // sass
yarn add sass-resources-loader // 配置全局scss变量
```
---
>> 注意: 如果想生成webpack.config.js请使用命令 `yarn eject`
